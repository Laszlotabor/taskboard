Welcome to taskMaster

TaskBoard Pro is a productivity and collaboration tool inspired by Trello. It allows users to organize projects into boards, manage tasks in customizable lists, and track progress using movable cards.

  • Boards: Create and manage multiple boards to organize different projects.
  • Lists: Add, edit, and reorder lists within boards to represent stages or categories.
  • Cards: Create detailed task cards, assign positions, and move them between lists using drag-and-drop.
  • Persistence: All changes are saved to a backend, ensuring data consistency across sessions.
  • User Authentication: Secure login and registration system to manage personal boards and tasks.
